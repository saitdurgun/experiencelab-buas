# BeltApp
**Skin Conductance Data Cleaning Tool**

BeltApp is a software tool developed by the Experience Lab at Breda University of Applied Sciences for segmenting, cleaning, and analysing physiological skin conductance (EDA/GSR) data for experience research.

---

## What Does BeltApp Do?

BeltApp enables researchers to:
- Record and segment biosignals such as skin conductance (SC/EDA)
- Automatically detect artifacts in physiological data
- Clean data using three replacement methods (Linear, Spline, NaN)
- Export cleaned data for further analysis

BeltApp uses a **deconvolution method** to separate tonic and phasic components of the skin conductance signal, allowing you to see true arousal responses more clearly.
